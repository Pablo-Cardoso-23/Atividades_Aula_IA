# 🧠 Inteligência Artificial - Atividades Acadêmicas

Este repositório foi criado para armazenar e organizar todos os projetos, scripts e atividades práticas desenvolvidas durante as aulas da disciplina de **Inteligência Artificial**.

O objetivo aqui é documentar a minha evolução na matéria, desde a exploração inicial de bases de dados até o treinamento e avaliação de modelos de Machine Learning.

## 🛠️ Tecnologias e Bibliotecas Utilizadas
Ao longo das atividades, as seguintes ferramentas estão sendo exploradas:
* **Linguagem:** Python
* **Manipulação de Dados:** Pandas, NumPy
* **Machine Learning:** Scikit-Learn
* **Visualização de Dados:** Plotly
